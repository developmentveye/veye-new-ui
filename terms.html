Cookies

A HTTP cookie (also called web cookie, Internet cookie, browser cookie, or simply cookie) is a small piece of data sent from a website and stored on the user's computer by the user's web browser while the user is browsing. Cookies store helpful information and add to a user’s convenience when they revisit a website but you must know what information is being stored in the cookie to build trust. We confirm that we do not use cookies on our website to store or capture any Sensitive Personal Information. We use them only to gather data around user access to various sections or pages of our website for eg. How many hits does each section or page gets etc. This data supports our web analytics to further enhance user experience. In addition, our e-mails may include a Tracking tag (also called Web Beacons or Web Bugs) to enable us to gather data around how many users accessed our emails. For an enhanced user experience, this tag also tracks which link in the e-mail you have clicked on to enable us to capture your area of interest and customize the information that you receive from us. The information collected by our Tracking tags is limited to, time and date of a page being viewed, and a description of the URL on which the tracking tag resides. These tags can be restricted if you don’t want to receive them via e-mail by disabling or refusing HTML format e-mails. You can opt to receive Text only e-mails instead.
